# AI-Assisted Code Analysis Tool

An AI-assisted code analysis tool that integrates a Python-based MCP server with Claude Desktop to evaluate backend code for vulnerabilities, code quality issues, and inefficiencies.

This project enables faster debugging, reduces manual code review effort, and improves overall code reliability by incorporating AI-driven analysis into the development workflow.

---

## Features

* Automated detection of code vulnerabilities and potential security risks
* Identification of code smells and inefficient logic patterns
* AI-assisted debugging and code quality analysis using Claude
* Reduced manual code review effort through automated workflows
* Improved developer productivity with faster feedback cycles

---

## 🏗️ Architecture Overview

```
Local Codebase → Python MCP Server → Claude Desktop → Analysis Output
```

### Flow:

1. Code is sent from the local project to the MCP server
2. MCP server processes and forwards structured input to Claude Desktop
3. Claude analyzes the code for:

   * Vulnerabilities
   * Code quality issues
   * Optimization opportunities
4. Results are returned and used for debugging and improvements

## ⚙️ Setup & Installation

### 1. Clone the repository

```bash
git clone https://github.com/manigandan0007/AI-Assisted-Code-Analysis-Tool.git
```

### 2. Create virtual environment

```bash
python -m venv venv
source venv/bin/activate   # Linux / Mac
venv\Scripts\activate      # Windows
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Configure MCP server

* Ensure MCP server is properly set up
* Configure connection to Claude Desktop
* Update any required environment variables
